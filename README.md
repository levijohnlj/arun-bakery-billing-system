# Arun Bakery Admin Dashboard

A minimalistic, high-performance bakery billing and inventory dashboard built with React, Vite, and Supabase.

## Features
- Real-time billing and sales tracking
- Inventory management
- Supabase backend for secure database storage

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up your `.env` variables for Supabase (`VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`).
3. Run the app:
   ```bash
   npm run dev
   ```
